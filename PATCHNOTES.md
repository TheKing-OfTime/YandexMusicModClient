## Что нового
- Улучшена функция расчёта энергии трека для анимации Моей Волны.
- Немного улучшен визуал превью в панели задач.
- Реализован миниплеер!
  - Включается по нажатию на иконку миниплеера в тайтлбаре.
  - Позволяет управлять воспроизведением, не разворачивая основное окно.
  - Пока что:
    - Не имеет возможность менять громкость.
    - Не учитывает параметры улучшения панели плеера.
    - Не учитывает тему приложения. (Всегда тёмная);
    - Не отображает временные метки трека. Только прогресс бар.
    - Нет возможности запустить окно без режима "всегда сверху".
  - Вышеописанные ограничения будут исправлены в будущих обновлениях.
- Добавлены новые команды в горячие клавиши:
  - `SET_PROGRESS|value` - установить прогресс воспроизведения на указанное значение (в минутах).
  - `SET_VOLUME|value` - установить громкость воспроизведения на указанное значение (в процентах).
